# 🐾 PawPaw

PawPaw es un mini proyecto de IA diseñado para interpretar comportamientos de mascotas y generar vídeos narrados de forma automática.

## Funcionalidades principales

### 1. PawTalk
Subes un vídeo de tu mascota y el sistema puede:
- detectar el comportamiento
- sugerir posibles interpretaciones
- confirmar el comportamiento mediante interacción por voz
- generar un vídeo narrado con subtítulos y audio

### 2. PawCare
Permite describir síntomas o comportamientos extraños de la mascota para obtener una orientación inicial antes de acudir al veterinario.

### 3. PawTrack
Módulo pensado para registrar rutinas, alimentación, hábitos de salud y patrones de comportamiento diario.

### 4. PawMemo
Módulo pensado para guardar recuerdos, hitos importantes e información personal de la mascota.

## Tecnologías utilizadas

- Python
- Gradio
- Whisper
- OpenCV
- Text-to-Speech
- Generación automática de subtítulos
- Sugerencia de comportamiento basada en IA

---

## Dataset

Este proyecto ha sido entrenado y probado con vídeos privados de mascotas recopilados personalmente.

Por motivos de privacidad, el dataset original **no se publica** en este repositorio.

El objetivo principal de este proyecto es mostrar el flujo funcional de una aplicación de IA para interacción y análisis de comportamiento animal.

---

## Objetivo del proyecto

El objetivo de PawPaw no es solo clasificar comportamientos de mascotas, sino transformar esa interpretación en una experiencia más interactiva, útil y emocional para el usuario.

Este proyecto combina:
- reconocimiento de comportamiento
- interacción por voz
- generación de texto
- subtitulado automático
- creación de vídeo

en un único prototipo funcional.
